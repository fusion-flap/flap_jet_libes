@apd_map.txt
The file contains the mapping of the APDCAM in a definite discharge range. The columns are the following:
  1. Name of the signal based on pixel number, i. e. BES-[1..4]-[1..8], e. g. BES-2-7.
  2. The fibre matrices of the input fibres of the light spliting optics setup.
  3. The torus hall fibre numbers, i. e. KY6-X, e. g. KY6-21. Empty means nothing is connected to that APDCam channel.
  4. Track number on spectrometer. It is 0 if the fibre is not connected to the spectrometer. (May be wrong but connection marking is reliable.)
  5. ADC-JPF-LPF number [001..032]. Names in get_rawsignal.pro: BES-ADC01,JPF/DH/KY6D-DOWN:001,JPF/DH/KY6D-FAST:001.
    Warning! The first zero should be removed from this string if ADC name is used.
